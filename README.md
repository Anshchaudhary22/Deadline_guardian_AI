#  Deadline Guardian AI

An intelligent task and deadline management system designed to help users organize work, track deadlines, and improve productivity.  
The system focuses on simplifying task management with a clean interface and structured workflow.



##  Overview

**Deadline Guardian AI** is a productivity-focused application that allows users to efficiently manage their daily tasks and deadlines. It helps reduce missed submissions and improves time management through organized task tracking.



##  Key Features

-  Add, update, and delete tasks easily  
-  Deadline tracking with clear status indicators  
-  Organized task dashboard for better visibility  
-  Filter and manage tasks efficiently  
-  Persistent storage support (local/database depending on implementation)  
-  Lightweight and fast performance  



##  Tech Stack

- Language: Java Script,HTML,CSS 
- Concepts Used: OOP (Object-Oriented Programming), Collections Framework  
- Data Handling: File Handling (FileReader, FileWriter)  
- Tools: VS Code , GitHub
